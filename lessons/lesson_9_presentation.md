# Power BI

---
# Что такое Power BI?

**Power BI** — инструмент для визуального представления данных
- Интерактивные отчёты и дашборды
- Работа через графический интерфейс
- Без программирования

**Power BI это финальный этап аналитического процесса**

---
# Логика работы

```
Данные → Модель → Визуализация → Интерпретация
```

1. Загрузка данных (CSV, Excel, БД)
2. Связывание таблиц
3. Построение графиков
4. "Data story telling"

---
# Power Query

Среда для подготовки данных:
- Удаление пропусков
- Изменение типов данных
- Разбиение столбцов
- Объединение таблиц


---
# Measures

Формулы для агрегированных значений:
- Сумма продаж
- Средняя зарплата
- Количество клиентов
- Процент роста


---
# Типы визуализаций

| Тип | Назначение |
|-----|------------|
| Bar/Column Chart | Сравнение категорий |
| Line Chart | Изменение во времени |
| Pie Chart | Доли от целого |
| Scatter Plot | Связь между переменными |
| Table/Matrix | Детальные данные |
| Card | Ключевые метрики |
| Map | Географическое распределение |

---
# Источники данных

- Файлы: Excel, CSV, JSON
- Базы данных: SQL Server, MySQL, PostgreSQL
- Облачные сервисы: Azure, Google Analytics
- Веб: API
- Python/R: интеграция скриптов

---
# Процесс работы

1. Get Data - загрузить данные
2. Transform Data - очистить в Power Query
3. Model - связать таблицы
4. Visualize - создать графики
5. Publish - поделиться отчётом

---
# Основные термины

| Термин | Определение |
|--------|-------------|
| Dashboard | Набор визуализаций на одной странице |
| Report | Документ с несколькими страницами |
| Dataset | Набор данных в Power BI |
| Power Query | Редактор подготовки данных |
| DAX | Язык формул для вычислений |
| Measure | Вычисляемая метрика |
| Slicer | Фильтр на странице |

---
# Практическое задание

1. Установить Power BI Desktop
2. Загрузить датасет 
3. Создать дашборд:
   - Столбчатая диаграмма
   - Линейный график 
   - Один фильтр (slicer)
4. Сохранить файл как .pbix
