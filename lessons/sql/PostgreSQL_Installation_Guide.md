# Установка и настройка PostgreSQL и pgAdmin

## Что такое PostgreSQL и pgAdmin?

- **PostgreSQL** - это мощная объектно-реляционная система управления базами данных (СУБД) с открытым исходным кодом
- **pgAdmin** - это графический интерфейс для работы с PostgreSQL

## Установка PostgreSQL и pgAdmin на Windows

### Шаг 1: Скачивание

1. Перейдите на официальный сайт: https://www.postgresql.org/download/windows/
2. Нажмите на ссылку **"Download the installer"**
3. Выберите последнюю версию для Windows (например, PostgreSQL 16.x)
4. Скачайте установщик для вашей системы (обычно Windows x86-64)

### Шаг 2: Установка

1. Запустите скачанный установщик (postgresql-16.x-windows-x64.exe)
2. Нажмите **"Next"** на приветственном экране
3. Выберите папку установки (по умолчанию `C:\Program Files\PostgreSQL\16`)
4. Выберите компоненты для установки:
   -  PostgreSQL Server (основной сервер)
   -  pgAdmin 4 (графический интерфейс)
   -  Stack Builder (менеджер дополнений)
   -  Command Line Tools (инструменты командной строки)
5. Выберите папку для данных (по умолчанию `C:\Program Files\PostgreSQL\16\data`)
6. Введите пароль для суперпользователя `postgres`
7. Выберите порт (по умолчанию **5432**)
8. Выберите локаль (рекомендуется оставить `Default locale`)
9. Нажмите **"Next"** и дождитесь завершения установки

### Шаг 3: Проверка установки

1. Откройте **Пуск** → найдите **pgAdmin 4**
2. Запустите pgAdmin 4
3. При первом запуске установите мастер-пароль для pgAdmin (можно использовать тот же, что и для postgres)

## Первое подключение к PostgreSQL

### Через pgAdmin 4

1. Откройте **pgAdmin 4**
2. В левом меню разверните **Servers**
3. Нажмите на **PostgreSQL 16** (или вашу версию)
(Возможно дополнительно будет необходимо ввести - )
4. Введите пароль, который вы создали при установке

### Структура в pgAdmin

```
Servers
└── PostgreSQL 16
    ├── Databases
    │   └── postgres (системная БД по умолчанию)
    ├── Login/Group Roles
    │   └── postgres (суперпользователь)
    └── Tablespaces
```

## Создание первой базы данных

### Через графический интерфейс

1. В pgAdmin разверните **Servers → PostgreSQL 16**
2. Правый клик на **Databases**
3. Выберите **Create → Database**
4. Введите имя базы данных: `name_db`
5. Владелец (Owner): `postgres`
6. Нажмите **Save**
