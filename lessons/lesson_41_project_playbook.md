# Варианты итоговых проектов 


### 1) EDA + базовая модель цены недвижимости
- Роль: Data Analyst / Junior DS
- Бизнес-вопрос: какие факторы сильнее всего влияют на цену жилья?
- Что делаем: очистка, EDA, корреляции, линейная/множественная регрессия.
- Стек: Pandas, Seaborn, Matplotlib, Scikit-learn.
- Результат: Notebook + краткий бизнес-вывод.

### 2) SQL + BI дашборд по продажам
- Роль: BI Analyst
- Бизнес-вопрос: какие товары, сегменты и периоды дают основной вклад в выручку?
- Что делаем: SQL-агрегации, витрина метрик, дашборд KPI.
- Стек: SQL, Power BI.
- Результат: интерактивный дашборд для менеджмента.

### 3) Мониторинг показателей через API
- Роль: Data Analyst
- Бизнес-вопрос: как меняются ключевые показатели во времени?
- Что делаем: загрузка данных по API, валидация, графики динамики.
- Стек: Requests, Pandas, визуализация.
- Результат: аналитический отчет или mini-dashboard.

### 4) Анализ вакансий по навыкам (Selenium)
- Роль: Data Analyst
- Бизнес-вопрос: какие навыки чаще всего требуются в выбранной роли?
- Что делаем: сбор данных, очистка, частотный анализ навыков.
- Стек: Selenium, Pandas, Seaborn/Matplotlib.
- Результат: карта востребованных навыков.


### 5) Прогноз оттока клиентов 
- Роль: Product Analyst / Data Scientist
- Бизнес-вопрос: кого удерживать в первую очередь?
- Что делаем: EDA, подготовка признаков, логистическая регрессия/деревья/RF, оценка ROC-AUC/F1.
- Стек: Pandas, Scikit-learn, визуализация.
- Результат: список high-risk клиентов + рекомендации.

### 6) A/B тест: анализ и продуктовое решение
- Роль: Product Analyst
- Бизнес-вопрос: стоит ли выкатывать новую фичу?
- Что делаем: проверка данных, uplift, p-value, доверительные интервалы, вывод по запуску.
- Стек: SQL, Pandas, SciPy.
- Результат: decision-report по эксперименту.

### 7) Прогноз продаж (Time Series)
- Роль: BI Analyst / Data Scientist
- Бизнес-вопрос: сколько товара нужно на ближайший период?
- Что делаем: тренд/сезонность, baseline, модель прогноза, MAE/RMSE.
- Стек: Pandas, модели временных рядов, визуализация.
- Результат: прогноз 

### 8) Кредитный скоринг (классификация)
- Роль: Data Scientist / Data Analyst
- Бизнес-вопрос: как снизить риск дефолта?
- Что делаем: подготовка данных, бинарная классификация, оценка precision/recall/F1.
- Стек: Pandas, Scikit-learn, XGBoost (если уверенно).
- Результат: скоринговая таблица и правила принятия решения.

### 9) Кластеризация клиентов (KMeans)
- Роль: Data Scientist / Product Analyst
- Бизнес-вопрос: какие есть устойчивые группы клиентов по поведению?
- Что делаем: выбор признаков, масштабирование, KMeans, интерпретация кластеров.
- Стек: Pandas, Scikit-learn, визуализация.
- Результат: профили сегментов + бизнес-рекомендации.

### 10) ML-модель + Streamlit интерфейс
- Роль: Data Scientist
- Бизнес-вопрос: как дать менеджеру простой интерфейс для прогнозов?
- Что делаем: обучаем модель из пройденных алгоритмов, оборачиваем в Streamlit.
- Стек: Scikit-learn, Pandas, Streamlit.
- Результат: рабочее приложение для предсказаний.

# Рекомендуемые источники данных 

## Публичные API для интеграции данных в реальном времени


* **Open-Meteo API:** Предоставляет бесплатный доступ к высокоточным историческим архивам погоды и глобальным прогнозам.
* **World Bank API / IMF Data API:** Фундаментальные базы макроэкономических показателей (ВВП, инфляция, демографические индексы) по всему миру.
* **Финансовые API (Alpha Vantage, Marketstack, Yahoo Finance):** Предоставляют гранулярные исторические и потоковые данные по котировкам акций, валютным парам и криптовалютам.
* **WHO Global Health Observatory:** Официальные глобальные данные по эпидемиологии и системам здравоохранения для создания социально-значимых аналитических проектов и дашбордов.

---

## Глобальные репозитории (для поиска массивных сырых датасетов)

* **Google Dataset Search:** Ультимативный агрегатор, индексирующий более 25 миллионов наборов данных, разбросанных по государственным порталам, академическим институтам и частным репозиториям.
